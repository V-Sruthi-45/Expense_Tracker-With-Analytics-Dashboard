# 💰 Expense Tracker with Analytics Dashboard

A modern expense tracking application with powerful analytics and spending insights. Track, analyze, and visualize your personal finances with beautiful charts and smart insights.

## ✨ Features

- 📊 **Analytics Dashboard** - Interactive charts and spending insights
- 💳 **Expense Management** - Add, edit, delete expenses with categories
- 🎨 **Modern UI** - Glossy dark theme with responsive design
- 🔐 **User Authentication** - Login, signup, and profile management
- 📱 **Mobile Friendly** - Works seamlessly on all devices
- 💾 **Data Export** - Export your expense data as JSON
- ⚙️ **Customizable Settings** - Currency, language, and preferences

## 🚀 Quick Start

### Prerequisites
- Node.js (v16+)
- Python 3 (for frontend server)

### Installation
```bash
# Clone the repository
git clone <repository-url>
cd expense-tracker-analytics

# Start backend server
node mock-server.js

# Start frontend server (in another terminal)
python -m http.server 8080
```

### Access the App
- 🌐 **Frontend**: http://localhost:8080
- 🔧 **Backend API**: http://localhost:5000

## 🛠️ Tech Stack

**Backend**
- Node.js with Express

**Frontend**
- HTML5, CSS3, JavaScript
